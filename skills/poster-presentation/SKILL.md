# Poster & Presentation — Conference Communication for Scientists

## Scientific Poster Layout

### Standard Dimensions
| Conference | Size | Orientation |
|------------|------|-------------|
| ASCO | 48" × 36" (4' × 3') | Landscape |
| AACR | 48" × 36" | Landscape |
| CAP | 48" × 36" | Landscape |
| AMP | 36" × 48" | Portrait (check annually) |
| Generic US | 48" × 36" | Landscape |
| Generic EU | A0 (841 × 1189 mm) | Portrait |

### Poster Section Layout (48" × 36" Landscape)
```
┌──────────────────────────────────────────────────────────┐
│                    TITLE BAR (full width)                 │
│  Title · Authors · Affiliations · Contact · Logo(s)      │
├──────────────┬──────────────┬──────────────┬─────────────┤
│              │              │              │             │
│ BACKGROUND   │   METHODS    │   RESULTS    │ CONCLUSIONS │
│              │              │   (widest)   │             │
│ • Problem    │ • Study      │ • Key figure │ • Summary   │
│ • Gap        │   design     │   (ROC/KM/   │   (3-4 pts) │
│ • Hypothesis │ • Assay      │    volcano)  │             │
│              │   protocol   │ • Table with │ FUTURE      │
│              │ • Stats plan │   primary    │ DIRECTIONS  │
│              │              │   endpoint   │ • Next step │
│              │              │ • Secondary  │             │
│              │              │   figures    │ REFERENCES  │
│              │              │              │ (5-8 max)   │
│              │              │              │             │
│              │              │              │ QR CODE     │
│              │              │              │ (to paper/  │
│              │              │              │  preprint)  │
├──────────────┴──────────────┴──────────────┴─────────────┤
│ ACKNOWLEDGMENTS · Funding (grant #) · Disclosures        │
└──────────────────────────────────────────────────────────┘
```

### Typography & Design Rules
```
Font hierarchy:
  Title:        72-96 pt, bold (readable from 10 feet)
  Section heads: 36-48 pt, bold
  Body text:     24-28 pt (readable from 4 feet)
  Captions:      20-24 pt
  References:    18-20 pt

Font choice: Arial, Helvetica, or Calibri (sans-serif only for posters)
Line spacing: 1.2-1.5x

Colors:
  - Maximum 3-4 colors total (background + 2-3 accent)
  - White or very light background (easier to read, cheaper to print)
  - Dark text (#333333, never pure black)
  - Use institution brand colors for accents
  - Color-blind safe for all data figures (Okabe-Ito palette)

Layout:
  - Column-based flow (left to right, top to bottom)
  - Margins: 1-2 inches from edges
  - Consistent spacing between sections
  - Figures > text (60/40 ratio — visual dominance)
  - White space is intentional — do not fill every gap
```

### Content Guidelines
```
Word count: 300-800 words total (less is more)
  Background:  50-100 words (2-3 sentences, one figure at most)
  Methods:     100-150 words (flowchart preferred over prose)
  Results:     150-300 words (figures carry the weight, minimal text)
  Conclusions: 50-100 words (3-5 bullet points max)

Figures:
  - Hero figure: the single most important result (ROC curve, KM plot, key comparison)
  - 300 DPI minimum, embedded (not linked)
  - Figure text must be legible at poster scale (≥18 pt)
  - Every figure needs a label (Figure 1, 2...) and brief caption
```

## 3-Minute Poster Pitch

### Structure (for poster walk judges)
```
Opening (15 sec):
  "Hi, I'm [name] from [institution]. We developed a [type] assay
   for [clinical application] — here's what we found."

Problem (30 sec):
  Point to Background section. State unmet need + one key statistic.
  "[Disease] affects [N] patients annually, and current [test] misses [X]%."

Approach (30 sec):
  Point to Methods. Describe assay in one sentence.
  "We measured [N biomarkers] in [specimen] from [N patients] using [platform]."

Key Result (60 sec):
  Point to hero figure. State primary endpoint.
  "Our panel achieved AUC of [X] with [Y]% sensitivity at [Z]% specificity,
   outperforming [comparator] (AUC [W])."
  Point to secondary results if time permits.

Significance (30 sec):
  Point to Conclusions.
  "This suggests [clinical implication]. Next, we're [validation plan]."

Close (15 sec):
  "Here's the QR code to our preprint. Happy to discuss further — any questions?"
```

## Conference Talk Slides (10-15 slides)

### Slide Structure
| Slide # | Content | Time |
|---------|---------|------|
| 1 | Title: Title, authors, affiliations, disclosures | 30 sec |
| 2 | Background: Clinical problem + unmet need | 1 min |
| 3 | Background: Current diagnostic landscape (what exists and its limitations) | 1 min |
| 4 | Hypothesis/Aims: One slide, 1-3 aims clearly stated | 30 sec |
| 5 | Methods: Study design diagram (flowchart — not paragraphs) | 1 min |
| 6 | Methods: Assay workflow (visual, minimal text) | 30 sec |
| 7 | Results: Primary endpoint — hero figure fills the slide | 1.5 min |
| 8 | Results: Secondary endpoint or subgroup analysis | 1 min |
| 9 | Results: Head-to-head comparison with existing standard | 1 min |
| 10 | Results: SHAP / explainability (if ML-based assay) | 30 sec |
| 11 | Discussion: Clinical implications (what this means for patients) | 1 min |
| 12 | Limitations + Future directions | 30 sec |
| 13 | Conclusions: 3-4 bullet points maximum | 30 sec |
| 14 | Acknowledgments: Funding, collaborators, disclosures | 15 sec |
| 15 | Questions: Contact info + QR code to paper/preprint | — |

### Slide Design Rules
```
Text:
  - Maximum 6 lines of text per slide (excluding title)
  - Title: 32-36 pt · Body: 24-28 pt · Footnotes: 18-20 pt
  - No full sentences — use keywords and phrases
  - Dark text on light background (white or very light grey)

Figures:
  - One figure per slide (fills 60-80% of slide area)
  - Figures must be self-explanatory with axis labels and legends
  - Animate builds sparingly (reveal data points, not flying text)

Template:
  - 16:9 aspect ratio (widescreen — standard for modern projectors)
  - Consistent header/footer with logo and slide number
  - Sans-serif font: Calibri, Arial, or Helvetica
  - Consistent color scheme (3 colors max: background + 2 accent)
```

## Graphical Abstract

### Structure (journal requirements)
```
Dimensions: typically 530 × 1200 pixels (varies by journal)
Resolution: 300 DPI
Format: TIFF, EPS, or high-quality PNG

Layout (left to right or top to bottom):
  Panel 1: Input (specimen type, patient population) — icon or simple illustration
  Panel 2: Method (assay workflow — simplified to 2-3 steps)
  Panel 3: Key result (single figure: ROC curve, bar chart, or comparison)
  Panel 4: Clinical implication (icon: patient outcome, clinical decision)

Connect panels with arrows (→) showing workflow direction

Design:
  - Maximum 5-6 colors total
  - Icons > clipart (use Biorender, Servier Medical Art, or Flaticon — cite source)
  - Minimal text: labels only, no sentences
  - Font: 10-14 pt at final size
  - No axis labels on embedded figures (too small) — just the visual pattern
```

## Export & Printing Checklist

```
Before submitting/printing:
- [ ] Dimensions match conference requirements exactly
- [ ] All text readable at viewing distance (4 feet for poster, screen for slides)
- [ ] Figures at 300 DPI minimum (check with: identify -verbose or PIL Image.info)
- [ ] Color-blind safe palette for all data figures
- [ ] Spell check (especially author names and gene symbols)
- [ ] Disclosure statement included (required by most conferences)
- [ ] QR code works (test with phone before printing)
- [ ] Export as PDF (poster) or PPTX (slides) — not just native format
- [ ] File size < 50 MB for upload (compress images if needed)
- [ ] Backup copy on cloud storage (not just USB)
```
